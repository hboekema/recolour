# Recolour

Project for recolouring grayscale images.

## Requirements

 - Python 3.5
 - Tensorflow 1.14

## Set-up

### Unix setup with virtualenv
 ```bash
 virtualenv venv
 source venv/bin/activate
 pip install -U pip
 deactivate
 source venv/bin/activate
 pip install -r requirements.txt
 ```

 or (may require permission)

 ```shell
 ./setup_venv.sh
 ```
