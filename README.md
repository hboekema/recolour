# Recolour

Project for recolouring grayscale images

## Requirements

 - Python 3.5
 - Tensorflow 1.14

## Set-up

### Unix Setup with virtualenv
 '''shell
 virtualenv venv
 source venv/bin/activate
 pip install -U pip
 deactivate
 source venv/bin/activate
 pip install -r requirements.txt
 '''

 or

 '''shell
 source setup_venv.sh
 '''
